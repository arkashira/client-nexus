## Getting Started

### Prerequisites

- Node.js
- npm or yarn
- PostgreSQL

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/arkashira/client-nexus.git
   ```
2. Navigate to the project directory:
   ```sh
   cd client-nexus
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Set up the database:
   ```sh
   psql -U postgres -f setup.sql
   ```
5. Start the development server:
   ```sh
   npm start
   ```

### Usage

1. Register a new account or log in.
2. Add your customers and their details.
3. Schedule appointments and manage tasks.
4. Use the analytics dashboard to gain insights.
